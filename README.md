# Village Civic Issue Tracker

A simple Python command-line project to track local civic problems in a village or town.

## Problem

Many local issues such as broken streetlights, potholes, drainage blockage, garbage piles, and water leakage are reported verbally but not tracked properly.

## Solution

This project allows users to record civic issues, assign priority, update their status, and view pending problems.

## Features

- Add a civic issue
- View all issues
- Update issue status
- Search issues
- Store issue data in a JSON file

## Project Goal

Build a meaningful real-world Python project while learning Git and GitHub from beginner to advanced level.

## Tech Stack

- Python
- JSON file storage
- Git
